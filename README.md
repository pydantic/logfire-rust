# logfire-rust

Rust SDKs for [Pydantic Logfire](https://pydantic.dev/logfire).

## Crates

| Crate | Description | crates.io |
|-------|-------------|-----------|
| `logfire` | Instrumentation SDK for sending traces/metrics/logs | [![crates.io](https://img.shields.io/crates/v/logfire.svg)](https://crates.io/crates/logfire) |
| `logfire-client` | Query client for reading data via SQL API | (not yet published) |
| `logfire-core` | Shared types (token parsing, region detection) | [![crates.io](https://img.shields.io/crates/v/logfire-core.svg)](https://crates.io/crates/logfire-core) |

See [DEVELOPMENT.md](DEVELOPMENT.md) for contributing and release instructions.
